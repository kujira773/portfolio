# 1. ポートフォリオをどのように開発したのか
## i.  初期のポートフォリオの問題
* webサイトの目的やターゲットを具体的に意識せずに作成してしまい、デザインの根拠について説明することができなかった
* コーディングやデザインについて迷うことが多く、作業が思うように進まなかった

## ii.　アジャイル方式の開発
上記の問題を解決するために、アジャイル方式一部取り入れました。

###  KPT
一人で作業する中で自発的にこまめな振り返りが必要だと考え、KPTを取り入れました。ポートフォリオや作業での良いところ(keep)・問題点(problem)・改善点(try)を洗い出します。

[作成したKPIはこちら]()　　

###  ユーザーストーリーとペルソナ　　
* webサイトの目的を明確するために、ユーザーストーリーを考えました。
* ユーザーのデザインの好みや要望に沿ったwebサイトをイメージしやすいように、ペルソナを設定しました。

###  カンバン
制作のスピードを上げるために作業ごとに優先順位をつけ、チケットで管理しました。

[作成したカンバンはこちら]()　　

## iii.　今後の課題　　
* KPTは最初problemばかりだったので、これからkeep(良い振り返り)も増やしていくこと
* カンバンでチケットを追加し忘れてるなど、上手く管理しきないところがあった
* 今回はユーザーからのフィードバックはなく、単に改善のサイクルを回す目的で作業していたので、エンドユーザーの視点が必要

## vi. 参考にした書籍・サイト
* [Webデザイン基礎入門](https://books.mdn.co.jp/books/3219203009/)、[HTML+CSS標準入門](https://books.mdn.co.jp/books/3219203017/)、[JavaScript基礎入門](https://books.mdn.co.jp/books/3219203012/)(エムディエヌコーポレーション)  
コーディングの基本を体系的に学習し、モバイルファーストのサイトのレイアウトやナビメニューのcssなどを参考にしました
* [未経験からプロのWebデザイナーになる！ 400レッスン以上の完全マスターコース](https://www.udemy.com/course/web-design-master/)  
photoshop, illustractorの基本的な操作を学び、画像の編集やイラストの作成を実際にポートフィリオで活かしました。
* [手を動かして学ぶプロダクトデザイン入門！デザイン思考・プロトタイピング・アジャイルの考え方と実践](https://www.udemy.com/course/vzhdbzks/)  
アジャイルのメリットが今回のポートフォリオ作成にも当てはまると考え、参考にしました。

# 2. ポートフォリオ:女性限定のシェアハウスのwebサイト

背景：入居者を増やすため、サイトをリニューアルすることになった

* 年齢：20歳
* 性別：女性
* 職業：都内に通学する大学生
* 性格：友達多い、新しいもの好き
* 趣味：語学、海外ドラマ、美容グッズ・コスメ好き

#### ペルソナ②
* 年齢：25歳
* 性別：女性
* 職業：都内に勤務する会社員

* 趣味：ヨガ・フィットネス、読書、買い物

#### ペルソナ③
* 年齢：30歳
* 性別：女性
* 職業：都内に勤務するフリーランス
* 性格：アクティブ、自由人
* 趣味：ランニング、ボルダリング、1人旅行

## iii. デザインで意識したポイント
* ライトグリーンをテーマカラーに・・・健康志向な女性のイメージ、新生活のフレッシュさ
* 利用者の口コミの写真をSNSのアイコン風にする・・・SNSを利用する若い人に共感を得る
* 申し込みボタンをページ全体に設置する・・・ページ戻りやボタンを探す煩わしさを解消する
* 家賃の情報より口コミやサービスの魅力を伝えるセクションを上部に置く・・・早い段階での離脱を防ぐ

# 3. ポートフォリオ:古民家風カフェのwebサイト

背景：店舗開店に合わせてサイトを立ち上げることになった  
目的：ショップカードやSNSからアクセスを増やし、来店者の口コミを増やす

## ii. ペルソナの設定
###### ※ヒアリングや調査・分析の元で設定したと仮定する

#### ペルソナ①
* 年齢：20歳
* 性別：女性
* 職業：都内に通学する大学生
* 性格：新しいもの好き、良い意味で広く浅い
* 趣味：写真、カフェ巡り、海外より国内旅行

#### ペルソナ②
* 年齢：25歳
* 性別：女性
* 職業：都内に勤務する会社員
* 性格：おとなしい、同じ趣味の友達がいる
* 趣味：漫画、お絵かき、猫やうさぎなど小動物を動物を飼っている

#### ペルソナ③
* 年齢：28歳
* 性別：男性
* 職業：都内に勤務する会社員
* 性格：多趣味、フォロワー多い、イベントに彼女と行く
* 趣味：読書、アニメ、鉄道、謎解き系、水族館・動物園、ディズニーよりジブリ、インスタよりTwitter

## iii. デザインで意識したポイント
* スライダーでメニューの写真を表示・・・メニューの写真を縦に並べないようにしてスクロールを少なくする
* 目立つところにSNSのシェアボタンを設置する・・・SNSで拡散してもらいやすくする
* テーマカラーを深いグリーン・ブラウンで統一・・・メニューの抹茶・ほうじ茶の渋さ・落ち着いたイメージ
